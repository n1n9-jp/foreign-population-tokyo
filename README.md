# 東京都における中国人人口の推移

インタラクティブなデータ可視化プロジェクト。東京都統計部から取得した外国人統計を基に、中国人人口の推移をデータ分析・可視化します。

## 📊 プロジェクト概要

このプロジェクトは、東京都における中国人人口の推移を1979年から2025年までの長期的なトレンドで分析し、インタラクティブなチャートを通じて可視化します。

**データ出典**: [東京都総務局統計部 - 外国人統計](https://www.toukei.metro.tokyo.lg.jp/gaikoku/ga-index.htm)

## 📁 作成されたデータファイル

|---------|--------|------|

| **china_population_timeseries.json** | 3.7K | 東京全体の時系列（1979-2025年、47年分） |

## ⚠️ 重要な注記

- **平成28年（2016年）まで**: 「中国」統計に台湾を含む
- **平成29年（2017年）以降**: 台湾は別扱い

データの比較分析を行う際には、この統計分類の変更点に注意が必要です。

## 📊 インタラクティブなチャート

**index.html** を開くと、D3.jsを使用したインタラクティブな折れ線グラフを表示できます。

### チャートの機能

- **自治体別の折れ線グラフ**: 1979-2025年の47年分のデータを可視化
- **凡例クリック機能**: 凡例の自治体名をクリックして表示/非表示を切り替え
- **「すべて表示」「すべて非表示」ボタン**: 一括操作が可能
- **ツールチップ**: マウスを折れ線の上に置くと、年と人口が表示
- **動的なラベル表示**: 表示中の自治体が15以下の場合、折れ線の右に自治体名を表示
- **レスポンシブデザイン**: パソコン、タブレット、スマートフォンに対応
- **動的な高さ調整**: 表示中の自治体数に応じてグラフの高さが自動調整

### 使い方

1. **ブラウザで開く**: index.htmlをブラウザで開く
2. **凡例で選択**: 右側の凡例で自治体をクリックして表示/非表示を切り替え
3. **「すべて表示」で全59自治体を表示** - 全体的なトレンドを把握できます
4. **「すべて非表示」で初期化** - すべての折れ線が非表示になります
5. **特定の自治体を比較**: 比較したい自治体のみを選択して凡例に表示

### デフォルト表示

デフォルトでは、2025年の中国人人口が多い**トップ15の自治体**を表示します。
